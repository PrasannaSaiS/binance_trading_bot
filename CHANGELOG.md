# Changelog

## [1.0.0] - 2025-10-23
### Added
- Initial release of Binance Futures Trading Bot for Testnet
- Support for Market, Limit, Stop-Market, and Stop-Limit orders
- Command-line interface for order placement and account management
- TWAP (Time Weighted Average Price) strategy implementation
- Grid Trading bot for automated range-bound trading
- Input validation, error handling, and detailed logging
